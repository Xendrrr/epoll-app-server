Приложение-сервер на C/C++ для GNU/Linux, который асинхронно обрабатывает клиентские подключения по протоколам TCP и UDP с помощью механизма мультиплексирования (на базе epoll) без использования сторонних библиотек.

Если сообщение клиента начинается с символа /, то оно интерпретируется как команда.  
В противном случае зеркалируется клиенту.  
Сервер должен поддерживать следующие команды:  
 
/time — возврат текущего времени и даты в формате "2025-11-10 17:28:45";  
/stats — возврат статистики (общее количество подключившихся клиентов и подключенных в данный момент);  
/shutdown — завершение работы.  

Для запуска сервера:  
g++ -std=c++23 -o server main.cpp  
./server 5252 (5252 -> любой желаемый порт)  

Для проверки работоспособности сервера (на localhost) можно использовать:  

TCP-протокол:  
  nc 127.0.0.1 5252  
  // здесь можно писать команды как в интерактивном режиме  
  
UDP-проткоол:
  nc -u 127.0.0.1 5252  
  // здесь можно писать команды как в интерактивном режиме
